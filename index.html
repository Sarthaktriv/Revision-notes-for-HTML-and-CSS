HTML

HTML (HyperText Markup Language) is the standard markup language for creating skeleton of web pages.

  1. <p>, <h1> to <h6>tag

    2. <p> tag is used to create paragraphs.<h1> to <h6> tags are used to create headings.

  2. <ul>, <ol>, <li> tag

    3. <ul> tag is used to create unordered list.<ol> tag is used to create ordered list.<li> tag is used to create list items.

  3. <div> tag

    4. <div> tag is used to create a container for other elements.

  4. <span> tag

    5. <span> tag is used to create a inline container for other elements.

  5. <a> tag

    6. <a> tag is used to create a hyperlink.

  6. <img> tag

    7. <img> tag is used to display images.

  7. <video> and <audio> tags

    8. <video> tag is used to display videos.<audio> tag is used to display audio.

  8. <form> tag

    9. <form> tag is used to create a form.

  9. <input> tag

    10. <input> tag is used to create input fields.

  10. <button> tag

    11. <button> tag is used to create buttons.

  11. <select> and <option> tags

    12. <select> tag is used to create a dropdown list.<option> tag is used to create options for the dropdown list.

  12. <table>,<tr>,<th> and <td> tags

    13. <table> tag is used to create tables.<tr> tag is used to create rows in a table.<th> tag is used to create header cells in a table.<td> tag is used to create data cells in a table.

  13. <iframe> tag

    14. <iframe> tag is used to embed another document within the current HTML document.

  14. <textarea> Tag

    15. This tag is used for take input in the form of paragraph or long text just like comment box.

*These tags have their own unique attributes which make them versatile for different styling and functional needs.

Semantic Tags

Semantic tags provide meaning to the structure of a webpage, making it more accessible and SEO-friendly.

  • <header> tag

    • <header> tag is used to define the header section of a webpage.

  • <footer> tag

    • <footer> tag is used to define the footer section of a webpage.

  • <main> tag

    • <main> tag is used to define the main content of a webpage.

  • <figure> and <figcaption> tags

    • <figure> tag is used to group media content like images or videos.<figcaption> tag is used to add a caption to the figure.

Note:

There are some tags which are now not recommended for use in modern HTML5 development. Like - <center>, <font>, <strike>, <big>, <br> etc.

CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML .

Fonts

CSS allows you to control the font family, size, weight, style, and other properties of text on a webpage.

  1. Font Family

    2. There are many types of fonts you can find on internet.

  2. Font Size

    3. The size of the font can be controlled using the font-size property.

  3. Font Weight

    4. The weight of the font can be controlled using the font-weight property.

  4. Font Style

    5. The style of the font can be controlled using the font-style property.

Colors

In CSS we can give color to our content using different ways.

  1. Color Keyword

    2. There are many color keywords that can be used in CSS.Like red, yellow, green, aqua etc.

  2. Hex code

    3. we can give color using Hex code.Like #ff0000 or any other code.

  3. RGB values

    4. we can give color using RGB values.Like rgb(255, 0, 0) or any other combination.

    All values lie between 0-255.

  4. RGBA values

    5. we can give color with transparency using RGBA values.Like rgba(255, 0, 0, 0.5) or any other combination.

    All values lie between 0-255 and the last value is for transparency which lies between 0-1.

  5. HSL values

    6. H - Hue (color)

    S - Saturation (intensity)

    L - Lightness (brightness)

    We can give color using HSL values.Like hsl(0, 100%, 50%) or any other combination.

Display Property

There are many display property values in CSS, such as block, inline, inline-block, and none.

  1. display:inline

    2. It is used to convert block elements to inline elements.

    Note:

     Some of the styles do not affect inline elements. Like width, margin, padding etc.

    In order to apply these styles, you can use the display property with the value "inline-block".

  2. display:flex

    3. It stands for Flexible Box Layout and is used to create flexible layouts.

    Many of the times it is used for centering content in the box . For this we use justify-content:center;

  3. display:block

    4. It is used to convert inline elements to block elements.

  4. display:none

    5. It is used to hide elements from the webpage.

    * If we only want to only hide the element with leaving the space . We can use:-

    visibility:hidden;

    It will hide the element and leave the space for it.

  5. display:grid

    6. It is used to converts an HTML element into a two-dimensional layout container, enabling you to organize content simultaneously into rows and columns.
